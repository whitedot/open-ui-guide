Checkbox vs Radio

Checkbox와 Radio의 차이점과 사용 시나리오입니다.

Checkbox

  • • 다중 선택 가능
  • • 독립적인 옵션들
  • • 전체 선택/해제 가능
  • • 예: 관심사, 동의 항목

Radio

  • • 단일 선택만 가능
  • • 상호 배타적 옵션
  • • 항상 하나는 선택됨
  • • 예: 성별, 결제 방법

Usage Guidelines

Do

  • • 여러 옵션 중 다중 선택이 필요할 때 사용하세요
  • • 라벨은 체크박스 오른쪽에 배치하세요
  • • 명확하고 간결한 라벨 텍스트를 사용하세요
  • • 관련 체크박스는 그룹으로 묶어 표시하세요
  • • 필수 항목에는 별표(*)를 표시하세요
  • • 충분한 터치 영역(44px 이상)을 확보하세요

Don't

  • • 단일 선택에 체크박스를 사용하지 마세요 (Radio 사용)
  • • 라벨 없이 체크박스만 사용하지 마세요
  • • 기본값으로 체크된 상태를 남용하지 마세요
  • • 너무 많은 체크박스를 한 화면에 나열하지 마세요
  • • 부정문으로 라벨을 작성하지 마세요
  • • disabled 상태를 기본값으로 사용하지 마세요
