Button

사용자 액션을 유도하는 인터랙티브 요소입니다. 다양한 변형과 상태를 제공합니다.

색상 사용 원칙

기본적으로 Base(Gray)를 사용하고, 주요 CTA(Call-to-Action)에만 Primary를 사용하세요.

  • • Base (Gray): 대부분의 일반 버튼에 사용 (권장 기본값)
  • • Primary (Violet): 화면당 1개의 주요 액션에만 사용

Variants

버튼의 시각적 스타일 변형입니다. Base와 Primary 각각 5가지 스타일을 제공합니다.

Base (권장 기본값)

Primary (주요 CTA에만 사용)

<button class="btn btn-md btn-primary-ghost">Ghost</button>

With Icons

아이콘과 함께 사용하는 버튼입니다. 고정 높이(h-10)를 사용하여 아이콘만, 아이콘+텍스트, 텍스트만 있는 버튼 모두 동일한 높이를 유지합니다.

높이 일관성 (h-10 = 40px)

HTML 
<!-- Text Only -->
<button class="btn btn-md btn-base-primary">Text</button>

<!-- Icon + Text -->
<button class="btn btn-md btn-base-primary btn-with-icon">
  <svg class="w-4 h-4">...</svg>
  Text
</button>

<!-- Icon Only (Square) -->

</button>

Text Links

페이지 이동이 필요한 경우 <a> 태그를 사용합니다. 패딩이나 마진 없이 텍스트에만 스타일을 적용합니다.

자세한 내용은 도움말 센터를 참고하세요. 또는 고객 지원팀에 문의할 수 있습니다.

Default Link Learn More External Link Muted Link
HTML 
<!-- Default Link -->
<a href="/page" class="link">Link Text</a>

<!-- Link with Arrow -->
<a href="/page" class="link link-with-icon">
  Learn More
  <svg class="w-4 h-4">...</svg>
</a>

<!-- External Link -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer" class="link link-with-icon">
  External Link
  <svg class="w-3.5 h-3.5">...</svg> <!-- External link icon -->
</a>

<!-- Muted Link -->
<a href="/page" class="link-muted">Muted Link</a>

Usage Guidelines

                    

                        

                            

                                

                                    
                                    Do
                                

                                
    
                                    
  • • Base(Gray)를 기본 버튼으로 사용하세요
    • 
                                    
  • • Primary 버튼은 화면당 하나의 주요 CTA에만 사용하세요
    • 
                                    
  • • 페이지 이동에는 <a> 태그를 사용하세요
    • 
                                    
  • • 버튼 텍스트는 명확한 액션 동사로 작성하세요
    • 
                                    
  • • 충분한 터치 영역(44px 이상)을 확보하세요
    • 
                                    
  • • 로딩 상태에서는 버튼을 비활성화하세요
    • 
                                

                            

                            

                                

                                    
                                    Don't
                                

                                
    
                                    
  • • 모든 버튼에 Primary 색상을 사용하지 마세요
    • 
                                    
  • • 한 화면에 여러 개의 Primary 버튼을 두지 마세요
    • 
                                    
  • • 페이지 이동에 <button>을 사용하지 마세요
    • 
                                    
  • • 위험한 액션에 Primary 색상을 사용하지 마세요
    • 
                                    
  • • 아이콘만 있는 버튼에 툴팁을 빠뜨리지 마세요
    • 
                                    
  • • disabled 상태에서 호버 효과를 주지 마세요
    •
