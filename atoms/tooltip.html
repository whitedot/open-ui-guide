Code Example

툴팁 구현 예제입니다.

HTML 
<!-- Basic Tooltip (Top) -->
<div class="relative group">
  <button class="px-4 py-2 bg-gray-100 rounded-lg">
    Hover me
  </button>
  <div class="absolute bottom-full left-1/2 -translate-x-1/2 mb-2
       px-3 py-1.5 bg-gray-900 text-white text-xs font-medium
       rounded-lg opacity-0 group-hover:opacity-100 group-focus-within:opacity-100
       transition-opacity duration-200 pointer-events-none whitespace-nowrap">
    Tooltip text
    <!-- Arrow -->
    <div class="absolute top-full left-1/2 -translate-x-1/2
         border-4 border-transparent border-t-gray-900"></div>
  </div>
</div>

<!-- Tooltip (Right) - For Compact Sidebar -->
<div class="relative group">
  <button class="p-3 rounded-lg text-gray-400 hover:bg-gray-100
         focus:bg-gray-100 focus:outline-none">
    <!-- Icon -->
  </button>
  <div class="absolute left-full top-1/2 -translate-y-1/2 ml-2
       px-3 py-1.5 bg-gray-900 text-white text-xs font-medium
       rounded-lg opacity-0 group-hover:opacity-100 group-focus-within:opacity-100
       transition-opacity duration-200 pointer-events-none whitespace-nowrap z-10">
    Menu name
    <!-- Arrow -->
    <div class="absolute right-full top-1/2 -translate-y-1/2
         border-4 border-transparent border-r-gray-900"></div>
  </div>
</div>

Usage Guidelines

Do

  • • 간결한 텍스트 사용 (1-2 단어 권장)
  • • 아이콘만 있는 버튼에 필수로 적용
  • • 적절한 위치에 표시 (가려지지 않도록)
  • • 적절한 지연 시간 적용 (200ms 권장)
  • • 터치 디바이스에서는 대체 방법 제공

Don't

  • • 긴 문장이나 설명 포함하지 않기
  • • 이미 명확한 버튼에는 불필요
  • • 필수 정보를 툴팁에만 두지 않기
  • • 클릭 가능한 요소를 툴팁 내에 두지 않기
  • • 너무 빠르게 나타나거나 사라지지 않도록
