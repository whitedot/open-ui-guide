Input

사용자로부터 텍스트 데이터를 입력받는 폼 요소입니다. 다양한 타입과 상태를 제공합니다.

Input Types

다양한 데이터 타입에 맞는 입력 필드입니다. 적절한 타입을 사용하면 모바일에서 최적화된 키보드가 표시됩니다.

HTML 
<input type="text" placeholder="..." class="w-full h-10 px-3 bg-white dark:bg-gray-900 border border-gray-300 dark:border-gray-600 rounded-lg text-gray-900 dark:text-white placeholder-gray-400 focus:ring-2 focus:ring-violet-500 focus:border-transparent">

<!-- 다른 타입들 -->
<input type="email" ...>
<input type="password" ...>
<input type="number" ...>
<input type="tel" ...>
<input type="url" ...>
<input type="search" ...>
<input type="date" ...>

States

입력 필드의 상호작용 상태입니다.

CSS Classes 
/* Hover */
hover:border-gray-400 dark:hover:border-gray-500

/* Focus */
focus:ring-2 focus:ring-violet-500 focus:border-transparent

/* Disabled */
disabled:opacity-50 disabled:cursor-not-allowed disabled:bg-gray-50

/* Read-only */
read-only:bg-gray-50 read-only:cursor-default

Validation States

폼 유효성 검사 결과를 시각적으로 표시합니다.

필수 항목입니다

확인되었습니다

보안이 취약할 수 있습니다

HTML 
<!-- Error State -->
<input class="... border-red-500 focus:ring-red-500">
<p class="text-sm text-red-600">에러 메시지</p>

<!-- Success State -->
<input class="... border-emerald-500 focus:ring-emerald-500">

<!-- Warning State -->
<input class="... border-amber-500 focus:ring-amber-500">

With Labels

레이블, 필수 표시, 도움말 텍스트가 포함된 입력 필드 패턴입니다.

8자 이상, 영문/숫자/특수문자 포함

HTML 
<!-- Basic Label -->
<div>
  <label class="block text-sm font-medium text-gray-700 mb-2">이름</label>
  <input type="text" ...>
</div>

<!-- Required Label -->
<label>이메일 <span class="text-red-500">*</span></label>

<!-- With Helper Text -->
<input ...>
<p class="mt-1.5 text-sm text-gray-500">도움말 텍스트</p>

With Icons

아이콘이 포함된 입력 필드입니다. 입력 내용의 맥락을 시각적으로 전달합니다.

HTML 
<!-- Leading Icon -->
<div class="relative">
  <svg class="absolute left-3 top-1/2 -translate-y-1/2 w-5 h-5 text-gray-400">...</svg>
  <input class="pl-10 pr-3 ...">
</div>

<!-- Trailing Icon -->
<div class="relative">
  <input class="pl-3 pr-10 ...">
  <svg class="absolute right-3 top-1/2 -translate-y-1/2 w-5 h-5 text-gray-400">...</svg>
</div>

Input Groups

접두사, 접미사, 버튼 등이 결합된 입력 그룹입니다.

https://
@gmail.com
HTML 
<!-- Prefix Text -->
<div class="flex">
  <span class="inline-flex items-center px-3 ... border-r-0 rounded-l-lg">https://</span>
  <input class="flex-1 ... rounded-r-lg">
</div>

<!-- With Button -->
<div class="flex">
  <input class="flex-1 ... rounded-l-lg">
  <button class="... rounded-r-lg">버튼</button>
</div>

Textarea

여러 줄의 텍스트를 입력받는 영역입니다.

0/200

HTML 
<!-- Default (resize-y) -->
<textarea rows="4" class="w-full px-3 py-2 ... resize-y"></textarea>

<!-- No Resize -->
<textarea class="... resize-none"></textarea>

<!-- Both directions -->
<textarea class="... resize"></textarea>

Usage Guidelines

                    

                        

                            

                                

                                    
                                    Do
                                

                                
    
                                    
  • • 항상 명확한 레이블을 함께 사용하세요
    • 
                                    
  • • 적절한 input type을 사용하여 모바일 키보드를 최적화하세요
    • 
                                    
  • • 필수 항목은 빨간 별표(*)로 표시하세요
    • 
                                    
  • • 에러 발생 시 구체적인 안내 메시지를 제공하세요
    • 
                                    
  • • 비밀번호 필드에는 보기/숨기기 토글을 제공하세요
    • 
                                    
  • • 충분한 터치 영역(44px 이상)을 확보하세요
    • 
                                

                            

                            

                                

                                    
                                    Don't
                                

                                
    
                                    
  • • placeholder를 레이블 대신 사용하지 마세요
    • 
                                    
  • • 입력 중에 실시간 유효성 검사를 과도하게 하지 마세요
    • 
                                    
  • • 불필요한 필수 항목을 만들지 마세요
    • 
                                    
  • • 에러 메시지를 빨간색 테두리만으로 표시하지 마세요
    • 
                                    
  • • 레이블 없이 아이콘만 사용하지 마세요
    • 
                                    
  • • 비밀번호 입력에 자동완성을 막지 마세요
    •
