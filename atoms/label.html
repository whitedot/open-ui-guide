Label

폼 요소와 연결되어 입력 필드의 목적을 설명하는 텍스트 요소입니다. 접근성과 사용성을 높이는 필수 컴포넌트입니다.

Basic Labels

기본 레이블 스타일입니다. for 속성으로 입력 필드와 연결합니다.

HTML 
<!-- Default Label -->
<label for="input-id" class="block text-sm font-medium text-gray-700 dark:text-gray-300 mb-2">
  레이블 텍스트
</label>

<!-- Bold Label -->
<label class="block text-sm font-semibold text-gray-900 dark:text-white mb-2">
  볼드 레이블
</label>

Sizes

레이블의 크기 변형입니다. 폼의 컨텍스트에 맞게 선택합니다.

Size Class Font Size Margin Bottom
xs text-xs 12px mb-1.5
sm text-sm 14px mb-2
base text-base 16px mb-2
lg text-lg 18px mb-2

Required & Optional

필수 항목과 선택 항목을 명확하게 표시합니다.

HTML 
<!-- Required with Asterisk -->
<label>이메일 <span class="text-red-500">*</span></label>

<!-- Required with Text -->
<label>비밀번호 <span class="text-red-500 text-xs font-normal">(필수)</span></label>

<!-- Optional with Text -->
<label>닉네임 <span class="text-gray-400 text-xs font-normal">(선택)</span></label>

<!-- Optional Badge Style -->
<label>
  전화번호
  <span class="ml-1.5 inline-flex items-center px-1.5 py-0.5 rounded text-xs font-medium bg-gray-100 text-gray-600">선택</span>
</label>

With Helper Text

레이블 아래 또는 입력 필드 아래에 추가 설명을 제공합니다.

8자 이상, 영문/숫자/특수문자 포함

http:// 또는 https://로 시작해야 합니다

설정 페이지에서 API 키를 발급받을 수 있습니다

동의합니다

이용약관 및 개인정보처리방침에 동의합니다

HTML 
<!-- Helper Below Label -->
<label class="block text-sm font-medium text-gray-700 mb-1">비밀번호</label>
<p class="text-xs text-gray-500 mb-2">8자 이상, 영문/숫자/특수문자 포함</p>
<input ...>

<!-- Helper Below Input -->
<label class="block text-sm font-medium text-gray-700 mb-2">웹사이트</label>
<input ...>
<p class="mt-1.5 text-xs text-gray-500">도움말 텍스트</p>

입력 필드의 상태에 따른 레이블 스타일 변화입니다.

올바른 형식으로 입력해주세요

확인되었습니다

Inline Labels

레이블과 입력 필드를 가로로 배치하는 패턴입니다.

8자 이상 입력하세요

HTML 
<!-- Basic Inline -->
<div class="flex items-center gap-4">
  <label class="w-24 text-sm font-medium text-gray-700 text-right flex-shrink-0">이름</label>
  <input class="flex-1 max-w-xs ...">
</div>

<!-- Inline with Helper (items-start for alignment) -->
<div class="flex items-start gap-4">
  <label class="... pt-2.5">비밀번호</label>
  <div class="flex-1 max-w-xs">
    <input ...>
    <p class="mt-1.5 text-xs text-gray-500">도움말</p>
  </div>
</div>

Floating Labels

입력 시 레이블이 위로 이동하는 플로팅 레이블 패턴입니다.

HTML 
<div class="relative">
  <input
    type="text"
    id="floating"
    placeholder=" "
    class="peer w-full h-12 px-3 pt-4 pb-1 ..."
  >
  <label
    for="floating"
    class="absolute left-3 top-1/2 -translate-y-1/2 text-gray-400 text-sm transition-all
           peer-placeholder-shown:top-1/2 peer-placeholder-shown:text-sm
           peer-focus:top-3 peer-focus:text-xs peer-focus:text-violet-600
           peer-[:not(:placeholder-shown)]:top-3 peer-[:not(:placeholder-shown)]:text-xs
           pointer-events-none"
  >
    이메일
  </label>
</div>

Usage Guidelines

Do

  • • 모든 입력 필드에 명확한 레이블을 제공하세요
  • • for 속성으로 입력 필드와 명시적 연결하세요
  • • 필수 항목은 별표(*) 또는 텍스트로 표시하세요
  • • 레이블은 짧고 명확하게 작성하세요
  • • 도움말이 필요한 경우 헬퍼 텍스트를 추가하세요
  • • 에러 발생 시 레이블 색상도 함께 변경하세요

Don't

  • • placeholder를 레이블 대신 사용하지 마세요
  • • 레이블 없이 입력 필드만 배치하지 마세요
  • • 모든 필드를 필수로 표시하지 마세요 (선택만 표시)
  • • 레이블 텍스트를 너무 길게 작성하지 마세요
  • • 입력 필드와 너무 멀리 배치하지 마세요
  • • 대문자만 사용하여 읽기 어렵게 만들지 마세요
