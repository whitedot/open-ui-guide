Badge vs Tag

Badge와 Tag의 차이점과 사용 시나리오입니다.

Badge

  • • 상태 표시 (성공, 오류, 대기 등)
  • • 알림 카운터
  • • 짧은 라벨 (1-2 단어)
  • • 주로 읽기 전용
  • • 다른 요소에 부착
완료 대기중 오류

Tag

  • • 콘텐츠 분류/카테고리
  • • 키워드, 해시태그
  • • 필터링 옵션
  • • 클릭/삭제 가능
  • • 독립적 사용
JavaScript React

Usage Guidelines

Do

  • • 짧은 텍스트만 사용하세요 (1-2 단어)
  • • 상태를 표현할 때 일관된 색상을 사용하세요
  • • 중요한 정보를 강조할 때 사용하세요
  • • 배지 수가 많을 때는 그룹화하세요
  • • 알림 배지는 새로운 정보가 있을 때만 표시하세요

Don't

  • • 긴 문장을 배지에 넣지 마세요
  • • 너무 많은 배지를 한 곳에 모으지 마세요
  • • 의미 없는 장식용으로 사용하지 마세요
  • • 같은 색상으로 다른 의미를 표현하지 마세요
  • • 항상 0인 알림 배지를 표시하지 마세요
