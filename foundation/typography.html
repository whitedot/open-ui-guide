Typography

일관된 타이포그래피 시스템으로 가독성과 시각적 계층을 구축합니다. 메인 서체는 Pretendard Variable을 사용합니다.

Font Family

Pretendard

Pretendard는 한글과 영문 모두 최적화된 가변 폰트입니다. 다양한 굵기와 스타일을 지원하며, 화면에서의 가독성이 뛰어납니다.

/* Font Stack */

'Pretendard Variable', 'Pretendard',

-apple-system, BlinkMacSystemFont,

system-ui, Roboto, 'Helvetica Neue',

Arial, sans-serif

특징

  • 가변 폰트 (Variable Font)
  • 한글/영문 최적화
  • 9가지 굵기 지원 (100-900)
  • 오픈소스 (OFL 라이선스)

Type Scale

시각적 계층을 위한 텍스트 크기 스케일입니다. Tailwind CSS 클래스를 사용합니다.

클래스 크기 미리보기 용도
text-xs 12px 가나다라마바사 ABCDEFG 라벨, 캡션
text-sm 14px 가나다라마바사 ABCDEFG 보조 텍스트
text-base 16px 가나다라마바사 ABCDEFG 본문 (기본)
text-lg 18px 가나다라마바사 ABCDEFG 강조 본문
text-xl 20px 가나다라마바사 ABCDEFG 섹션 제목
text-2xl 24px 가나다라마바사 ABCDEFG 서브 타이틀
text-3xl 30px 가나다라마바사 페이지 제목

Font Weight

텍스트의 시각적 강조를 위한 굵기 설정입니다.

가나다 Aa

font-normal 400

가나다 Aa

font-medium 500

가나다 Aa

font-semibold 600

가나다 Aa

font-bold 700

Text Colors

텍스트 계층을 위한 색상 시스템입니다. 라이트/다크 모드 모두 지원합니다.

Primary

주요 텍스트에 사용합니다

text-gray-900 dark:text-white
Secondary

보조 텍스트에 사용합니다

text-gray-600 dark:text-gray-400
Tertiary

힌트, 플레이스홀더에 사용합니다

text-gray-500 dark:text-gray-500
Accent

강조에 사용합니다

text-violet-600 dark:text-violet-400

Usage Guidelines

Do

  • • 명확한 시각적 계층 구조를 유지하세요
  • • 일관된 Type Scale을 사용하세요
  • • 본문은 text-base (16px)를 기본으로 사용하세요
  • • 다크모드에서 적절한 대비를 유지하세요
  • • 제목에는 font-semibold 이상을 사용하세요

Don't

  • • 너무 많은 크기를 혼합하지 마세요
  • • 본문에 text-xs (12px)를 사용하지 마세요
  • • 긴 텍스트에 font-bold를 과도하게 사용하지 마세요
  • • 대비가 낮은 색상 조합을 피하세요
  • • 커스텀 폰트 크기를 임의로 만들지 마세요
