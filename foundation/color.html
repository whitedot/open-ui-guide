Color

계층형 컬러 시스템으로 일관된 UI를 구축합니다. 기본 색상은 Gray 모노톤, 강조 색상은 Violet 계열을 사용합니다.

Color Hierarchy

Base

Gray Monotone - 배경, 텍스트, 테두리 등 기본 UI 요소

Primary

Violet - CTA 버튼, 링크, 활성 상태 등 강조 요소

Semantic

Success, Warning, Error, Info 상태 표시

Usage Guidelines

Do

  • • 기본 버튼은 Gray를 사용하세요
  • • 중요한 액션에만 Violet을 사용하세요
  • • 다크모드에서 적절한 대비를 유지하세요
  • • 의미에 맞는 Semantic 색상을 사용하세요

Don't

  • • Violet을 과도하게 사용하지 마세요
  • • 여러 강조 색상을 혼합하지 마세요
  • • 낮은 대비의 텍스트를 사용하지 마세요
  • • 색상만으로 정보를 전달하지 마세요
