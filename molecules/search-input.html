Search Input

검색 기능을 제공하는 복합 입력 컴포넌트입니다. 아이콘, 버튼, 자동완성 등 다양한 요소가 결합됩니다.

Basic Search

검색 아이콘이 포함된 기본 검색 입력 필드입니다.

HTML 
<!-- Basic Search -->
<div class="relative">
  <svg class="absolute left-3 top-1/2 -translate-y-1/2 w-5 h-5 text-gray-400">...</svg>
  <input type="search" class="w-full h-10 pl-10 pr-4 ... rounded-lg">
</div>

Search with Clear Button

입력 내용을 지울 수 있는 X 버튼이 포함된 검색 필드입니다.

HTML + JavaScript 
<div class="relative">
  <svg class="absolute left-3 ...">...</svg>
  <input type="search" class="pl-10 pr-10 ..." oninput="toggleClear()">
  <button class="absolute right-3 top-1/2 -translate-y-1/2 ..." onclick="clearSearch()">
    <!-- X Icon -->
  </button>
</div>

Search with Button

검색 버튼이 결합된 입력 필드입니다. 명시적인 검색 액션을 제공합니다.

HTML 
<!-- Attached Button -->
<div class="flex">
  <div class="relative flex-1">
    <svg class="absolute left-3 ...">...</svg>
    <input class="... rounded-l-lg">
  </div>
  <button class="... rounded-r-lg">검색</button>
</div>

Search with Category

카테고리 선택과 결합된 검색 필드입니다. 검색 범위를 좁힐 수 있습니다.

HTML 
<!-- Select + Input + Button -->
<div class="flex">
  <select class="... rounded-l-lg">...</select>
  <input class="flex-1 border-y ...">
  <button class="... rounded-r-lg">검색</button>
</div>

Search with Autocomplete

검색어 자동완성 드롭다운이 포함된 검색 필드입니다. 사용자 입력에 따라 제안이 표시됩니다.

Search States

검색 중, 로딩, 결과 없음 등 다양한 상태를 표현합니다.

검색 결과가 없습니다

Expandable Search

공간을 절약하기 위해 아이콘에서 확장되는 검색 필드입니다.

JavaScript 
function toggleExpandSearch() {
  const btn = document.getElementById('expandBtn');
  const search = document.getElementById('expandableSearch');

  btn.classList.toggle('hidden');
  search.classList.toggle('hidden');

  if (!search.classList.contains('hidden')) {
    search.querySelector('input').focus();
  }
}

Usage Guidelines

Do

  • • 검색 아이콘을 왼쪽에 배치하여 검색 필드임을 명확히 표시
  • • placeholder로 검색 가능한 항목을 안내
  • • 검색 중 로딩 상태를 시각적으로 표시
  • • 결과가 없을 때 명확한 메시지 제공
  • • 자동완성 결과에서 일치하는 부분 강조
  • • 키보드 네비게이션 지원 (위/아래 화살표)

Don't

  • • 너무 많은 카테고리 옵션을 한번에 표시
  • • 짧은 검색어에 대해 과도한 API 호출
  • • 검색 결과 없음 상태를 빈 화면으로 방치
  • • Clear 버튼 없이 긴 검색어 입력 요구
  • • 자동완성 결과가 너무 많아 스크롤이 길어짐
  • • 모바일에서 작은 터치 영역 사용
