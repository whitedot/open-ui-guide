Usage Guidelines

Do

  • • 홈페이지부터 현재 페이지까지 전체 경로를 표시하세요
  • • 현재 페이지는 클릭 불가능하게 표시하세요
  • • 페이지 상단에 일관된 위치에 배치하세요
  • • 5단계 이상일 경우 생략(...) 표시를 사용하세요
  • • aria-label을 사용하여 접근성을 확보하세요
  • • 명확한 구분자(/, >, 등)를 사용하세요

Don't

  • • 너무 긴 페이지 이름을 사용하지 마세요
  • • 현재 페이지를 클릭 가능한 링크로 만들지 마세요
  • • 단일 페이지 또는 홈에서는 사용하지 마세요
  • • 브레드크럼을 주요 내비게이션으로 사용하지 마세요
  • • 페이지 계층과 다른 경로를 표시하지 마세요
  • • 모바일에서 너무 많은 항목을 표시하지 마세요
