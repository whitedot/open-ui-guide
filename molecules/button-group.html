JavaScript 
// Single Select Toggle
function selectSingle(btn) {
    const group = btn.parentElement;
    group.querySelectorAll('button').forEach(b => {
        b.className = 'px-4 py-2 bg-white text-gray-700 font-medium border-r border-gray-200 hover:bg-gray-50 transition-colors';
    });
    btn.className = 'px-4 py-2 bg-violet-600 text-white font-medium border-r border-violet-500 transition-colors';
}

// Multi Select Toggle
function toggleMulti(btn) {
    const isSelected = btn.dataset.selected === 'true';
    btn.dataset.selected = (!isSelected).toString();
    if (!isSelected) {
        btn.className = 'px-4 py-2 bg-violet-600 text-white font-medium border-r border-violet-500 transition-colors';
    } else {
        btn.className = 'px-4 py-2 bg-white text-gray-700 font-medium border-r border-gray-200 hover:bg-gray-50 transition-colors';
    }
}

Usage Guidelines

Do

  • • 관련된 액션들을 논리적으로 그룹화하세요
  • • 버튼 간 일관된 크기와 스타일을 유지하세요
  • • 가장 중요한 액션을 시각적으로 강조하세요
  • • 토글 그룹에서 현재 선택 상태를 명확히 표시하세요
  • • 모바일에서 적절한 터치 영역을 확보하세요

Don't

  • • 관련 없는 액션을 같은 그룹에 넣지 마세요
  • • 그룹 내에 너무 많은 버튼을 넣지 마세요 (3-5개 권장)
  • • 버튼 텍스트를 너무 길게 작성하지 마세요
  • • Primary 스타일의 버튼을 한 그룹에 여러 개 사용하지 마세요
  • • 비활성화된 버튼만으로 그룹을 구성하지 마세요
