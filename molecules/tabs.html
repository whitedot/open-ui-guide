Code Example

기본 탭 구현 코드입니다.

HTML + JavaScript 
<!-- Tab Container -->
<div data-tabs="example">
  <!-- Tab List -->
  <div class="border-b border-gray-200">
    <nav class="flex gap-8">
      <button class="tab-btn active ..." data-tab="tab1">Tab 1</button>
      <button class="tab-btn ..." data-tab="tab2">Tab 2</button>
    </nav>
  </div>
  <!-- Tab Panels -->
  <div class="tab-panel active" data-panel="tab1">Content 1</div>
  <div class="tab-panel hidden" data-panel="tab2">Content 2</div>
</div>

<!-- JavaScript -->
<script>
document.querySelectorAll('[data-tabs]').forEach(container => {
  container.querySelectorAll('.tab-btn').forEach(btn => {
    btn.addEventListener('click', () => {
      // Update active tab button
      // Show corresponding panel
    });
  });
});
</script>

Usage Guidelines

Do

  • • 관련된 콘텐츠를 논리적으로 그룹화하세요
  • • 탭 레이블은 짧고 명확하게 작성하세요
  • • 현재 활성 탭을 시각적으로 명확히 표시하세요
  • • 탭 순서는 중요도나 사용 빈도를 고려하세요
  • • 모바일에서는 스크롤 가능한 탭을 사용하세요
  • • 키보드 네비게이션을 지원하세요

Don't

  • • 너무 많은 탭을 사용하지 마세요 (5-7개 이하 권장)
  • • 탭 레이블에 긴 텍스트를 사용하지 마세요
  • • 탭 간 콘텐츠 높이 차이가 크면 피하세요
  • • 중첩된 탭 구조를 사용하지 마세요
  • • 탭 전환 시 페이지 이동을 발생시키지 마세요
  • • 탭 콘텐츠가 비어있는 상태로 두지 마세요
