Code Example

기본 페이지네이션 구현 코드입니다.

HTML 
<nav class="flex items-center gap-1" aria-label="Pagination">
  <!-- Previous -->
  <button class="p-2 text-gray-400 cursor-not-allowed" disabled>
    <!-- Arrow Icon -->
  </button>

  <!-- Active Page -->
  <button class="w-10 h-10 ... bg-violet-600 text-white">1</button>

  <!-- Other Pages -->
  <button class="w-10 h-10 ... hover:bg-gray-100">2</button>
  <button class="w-10 h-10 ... hover:bg-gray-100">3</button>

  <!-- Ellipsis -->
  <span class="w-10 h-10 ... text-gray-400">...</span>

  <!-- Next -->
  <button class="p-2 hover:bg-gray-100 rounded-lg">
    <!-- Arrow Icon -->
  </button>
</nav>

Usage Guidelines

Do

  • • 현재 페이지를 시각적으로 명확히 표시하세요
  • • 비활성화된 버튼은 명확히 구분되게 하세요
  • • 총 페이지 수나 항목 수를 표시하세요
  • • 충분한 터치 영역(44px 이상)을 확보하세요
  • • 키보드 네비게이션을 지원하세요
  • • 페이지 변경 시 스크롤 위치를 맨 위로 이동하세요

Don't

  • • 너무 많은 페이지 번호를 한 번에 표시하지 마세요
  • • 첫 페이지에서 이전 버튼을 숨기지 마세요 (비활성화)
  • • 마지막 페이지에서 다음 버튼을 숨기지 마세요
  • • 페이지 번호가 적을 때 생략 부호를 사용하지 마세요
  • • 모바일에서 너무 많은 요소를 표시하지 마세요
  • • 페이지 로딩 중 인터랙션을 허용하지 마세요
