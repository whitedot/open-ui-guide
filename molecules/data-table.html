HTML 
<!-- Basic Table -->
<div class="rounded-xl border overflow-hidden">
  <table class="w-full">
    <thead class="bg-gray-50 border-b">
      <tr>
        <th class="px-6 py-3 text-left text-xs font-medium uppercase">이름</th>
        <th class="px-6 py-3 text-left text-xs font-medium uppercase">이메일</th>
        <th class="px-6 py-3 text-left text-xs font-medium uppercase">상태</th>
      </tr>
    </thead>
    <tbody class="divide-y">
      <tr class="hover:bg-gray-50">
        <td class="px-6 py-4 text-sm font-medium">홍길동</td>
        <td class="px-6 py-4 text-sm">hong@example.com</td>
        <td class="px-6 py-4">
          <span class="px-2.5 py-0.5 rounded-full text-xs bg-emerald-100">활성</span>
        </td>
      </tr>
    </tbody>
  </table>
</div>

Usage Guidelines

Do

  • • 컬럼 헤더는 간결하고 명확하게 작성
  • • 데이터 정렬은 컬럼 유형에 맞게 설정
  • • 많은 데이터에는 페이지네이션 적용
  • • 호버/선택 상태로 인터랙션 표시
  • • 모바일에서는 가로 스크롤 허용
  • • 빈 상태에는 적절한 안내 메시지 표시

Don't

  • • 너무 많은 컬럼으로 가독성 저하
  • • 헤더 없이 데이터만 나열
  • • 행 구분선 없이 데이터 표시
  • • 모바일에서 테이블 잘림 현상
  • • 액션 버튼을 너무 작게 만들기
  • • 로딩 상태 표시 없이 대기
