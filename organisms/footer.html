Sticky Footer Layout

콘텐츠가 적어도 푸터가 항상 화면 하단에 위치하는 레이아웃입니다.

Flexbox를 활용한 Sticky Footer

콘텐츠 양에 관계없이 푸터가 항상 뷰포트 하단에 위치합니다.

HTML Structure 
<body class="min-h-screen flex flex-col">
  <header>...</header>

  <main class="flex-1">
    <!-- 메인 콘텐츠 -->
  </main>

  <footer>
    <!-- 푸터는 자동으로 하단에 위치 -->
  </footer>
</body>

/* 핵심 클래스:
   - body: min-h-screen flex flex-col
   - main: flex-1 (남은 공간을 모두 차지)
   - footer: 자동으로 하단에 배치됨
*/

Usage Guidelines

Do

  • • 이용약관, 개인정보처리방침 등 법적 링크를 포함하세요
  • • 저작권 표시에 현재 연도를 사용하세요
  • • 링크를 논리적인 카테고리로 그룹화하세요
  • • 소셜 미디어 아이콘은 target="_blank"로 새 탭에서 열리게 하세요
  • • 모바일에서 링크 그룹이 잘 표시되는지 확인하세요
  • • Sticky footer 레이아웃으로 콘텐츠가 적어도 푸터가 하단에 위치하게 하세요

Don't

  • • 너무 많은 링크로 푸터를 복잡하게 만들지 마세요
  • • 중요한 CTA를 푸터에만 배치하지 마세요
  • • 헤더와 동일한 네비게이션을 푸터에 반복하지 마세요
  • • 소셜 아이콘 없이 텍스트 링크만 사용하지 마세요
  • • 뉴스레터 폼에 너무 많은 필드를 요구하지 마세요
  • • 다크 푸터에서 대비가 낮은 색상을 사용하지 마세요
