Modal

사용자의 주의를 집중시키고 중요한 정보나 액션을 제공하는 오버레이 컴포넌트입니다.

Anatomy

Modal은 배경 오버레이와 콘텐츠 영역으로 구성됩니다.

1. Backdrop (Overlay)
2. Header

Title

3. Body

Content area

4. Footer

1. Backdrop

배경 오버레이, 클릭 시 닫힘

2. Header

제목, 닫기 버튼

3. Body

주요 콘텐츠 영역

4. Footer

액션 버튼

Basic Modal

기본적인 모달 구조입니다. 제목, 내용, 액션 버튼을 포함합니다.

HTML 
<!-- Modal Backdrop -->
<div class="fixed inset-0 z-50 flex items-center justify-center">
  <!-- Overlay -->
  <div class="absolute inset-0 bg-black/50" onclick="closeModal()"></div>

  <!-- Modal Content -->
  <div class="relative bg-white dark:bg-gray-800 rounded-xl shadow-xl w-full max-w-md mx-4">
    <!-- Header -->
    <div class="flex items-center justify-between px-6 py-4 border-b border-gray-200 dark:border-gray-700">
      <h3 class="text-lg font-semibold text-gray-900 dark:text-white">Modal Title</h3>
      <button class="text-gray-400 hover:text-gray-600 ...">
        <!-- Close icon -->
      </button>
    </div>

    <!-- Body -->
    <div class="px-6 py-4">
      <p class="text-gray-600 dark:text-gray-400">Modal content goes here.</p>
    </div>

    <!-- Footer -->
    <div class="flex justify-end gap-3 px-6 py-4 border-t border-gray-200 dark:border-gray-700">
      <button class="px-4 py-2 text-gray-600 hover:bg-gray-100 rounded-lg">Cancel</button>
      <button class="px-4 py-2 bg-violet-600 hover:bg-violet-700 text-white rounded-lg">Confirm</button>
    </div>
  </div>
</div>

Size Variations

용도에 따라 다양한 크기의 모달을 사용할 수 있습니다.

Size Class Max Width Use Case
Small max-w-sm 384px 확인 다이얼로그, 경고
Medium max-w-md 448px 간단한 폼, 기본 모달
Large max-w-lg 512px 복잡한 폼, 상세 정보
Extra Large max-w-xl 576px 대형 콘텐츠, 미디어
2XL max-w-2xl 672px 테이블, 복잡한 레이아웃

Confirmation Modal

사용자의 확인이 필요한 중요한 액션에 사용합니다.

Delete Confirmation 
<div class="relative bg-white rounded-xl shadow-xl w-full max-w-sm mx-4">
  <div class="p-6 text-center">
    <!-- Icon -->
    <div class="w-12 h-12 mx-auto mb-4 rounded-full bg-red-100 dark:bg-red-900/30 flex items-center justify-center">
      <svg class="w-6 h-6 text-red-600">...</svg>
    </div>

    <h3 class="text-lg font-semibold text-gray-900 mb-2">Delete Item?</h3>
    <p class="text-gray-600 mb-6">This action cannot be undone.</p>

    <div class="flex gap-3">
      <button class="flex-1 px-4 py-2 border border-gray-300 rounded-lg ...">Cancel</button>
      <button class="flex-1 px-4 py-2 bg-red-600 hover:bg-red-700 text-white rounded-lg ...">Delete</button>
    </div>
  </div>
</div>

Form Modal

폼 입력을 위한 모달입니다.

Form Modal 
<div class="relative bg-white rounded-xl shadow-xl w-full max-w-md mx-4">
  <!-- Header -->
  <div class="px-6 py-4 border-b border-gray-200">
    <h3>Create New Item</h3>
  </div>

  <!-- Form Body -->
  <form class="px-6 py-4 space-y-4">
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Name</label>
      <input type="text" class="w-full px-3 py-2 border border-gray-300 rounded-lg focus:ring-2 focus:ring-violet-500">
    </div>
    <div>
      <label class="block text-sm font-medium text-gray-700 mb-1">Description</label>
      <textarea rows="3" class="w-full px-3 py-2 border border-gray-300 rounded-lg ..."></textarea>
    </div>
  </form>

  <!-- Footer -->
  <div class="flex justify-end gap-3 px-6 py-4 border-t border-gray-200">
    <button type="button">Cancel</button>
    <button type="submit">Create</button>
  </div>
</div>

Scrollable Modal

콘텐츠가 많을 때 스크롤이 가능한 모달입니다.

Scrollable Body 
<div class="relative bg-white rounded-xl shadow-xl w-full max-w-lg mx-4 max-h-[80vh] flex flex-col">
  <!-- Fixed Header -->
  <div class="flex-shrink-0 px-6 py-4 border-b border-gray-200">
    <h3>Terms of Service</h3>
  </div>

  <!-- Scrollable Body -->
  <div class="flex-1 overflow-y-auto px-6 py-4">
    <!-- Long content -->
  </div>

  <!-- Fixed Footer -->
  <div class="flex-shrink-0 px-6 py-4 border-t border-gray-200">
    <button>Accept</button>
  </div>
</div>

Fullscreen Modal

전체 화면을 사용하는 모달입니다. 모바일에서 유용합니다.

Fullscreen Modal 
<!-- Desktop: 일반 모달, Mobile: 전체화면 -->
<div class="relative bg-white shadow-xl
  w-full h-full                      /* Mobile: 전체화면 */
  md:w-auto md:h-auto md:max-w-lg md:mx-4 md:rounded-xl  /* Desktop: 일반 */">
  ...
</div>

<!-- 항상 전체화면 -->
<div class="fixed inset-0 z-50 bg-white flex flex-col">
  <!-- Header -->
  <div class="flex-shrink-0 h-14 px-4 border-b ...">...</div>
  <!-- Body -->
  <div class="flex-1 overflow-y-auto p-4">...</div>
</div>

Animation

모달 열기/닫기 애니메이션 패턴입니다.

Tailwind CSS Animation

Tailwind의 기본 transition과 transform 클래스를 조합하여 애니메이션을 구현합니다. JavaScript로 클래스를 토글하여 상태를 변경합니다.

Animation Classes 
/* Backdrop fade */
<div class="absolute inset-0 bg-black/50
  transition-opacity duration-300
  opacity-0"></div>       /* Closed state */

/* Modal scale + fade */
<div class="relative bg-white rounded-xl
  transition-all duration-300
  opacity-0 scale-95">              /* Closed state */
  ...
</div>

/* Open state: JavaScript로 클래스 토글 */
backdrop: 'opacity-0' → 'opacity-100'
modal: 'opacity-0 scale-95' → 'opacity-100 scale-100'

Accessibility

접근성을 위한 ARIA 속성과 키보드 네비게이션입니다.

ARIA Attributes 
<div
  role="dialog"
  aria-modal="true"
  aria-labelledby="modal-title"
  aria-describedby="modal-description"
>
  <h3 id="modal-title">Modal Title</h3>
  <p id="modal-description">Modal description text</p>
</div>

/* 키보드 인터랙션 */
- ESC: 모달 닫기
- Tab: 모달 내 포커스 순환 (focus trap)
- 모달 열릴 때 첫 번째 포커스 가능 요소로 포커스 이동
- 모달 닫힐 때 트리거 버튼으로 포커스 복원

Usage Guidelines

Do

  • • 중요한 액션에 대한 확인이 필요할 때 사용하세요
  • • 명확한 제목과 액션 버튼을 제공하세요
  • • ESC 키와 배경 클릭으로 닫을 수 있게 하세요
  • • 적절한 크기의 모달을 선택하세요
  • • 모달이 열릴 때 배경 스크롤을 잠그세요
  • • 접근성을 위한 ARIA 속성을 추가하세요

Don't

  • • 여러 개의 모달을 중첩하여 사용하지 마세요
  • • 불필요한 정보에 모달을 사용하지 마세요
  • • 닫기 버튼 없이 모달을 만들지 마세요
  • • 너무 많은 콘텐츠를 모달에 넣지 마세요
  • • 페이지 로드 시 자동으로 모달을 열지 마세요
  • • 삭제와 같은 위험한 액션에 Primary 색상을 사용하지 마세요
