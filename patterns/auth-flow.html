Usage Guidelines

Do

  • • 명확한 에러 메시지를 제공하세요
  • • 비밀번호 강도를 시각적으로 표시하세요
  • • 소셜 로그인 옵션을 제공하세요
  • • 로딩 상태를 명확하게 표시하세요
  • • 접근성을 고려한 폼을 만드세요
  • • 2단계 인증 옵션을 제공하세요

Don't

  • • 불필요한 정보를 요구하지 마세요
  • • 비밀번호를 평문으로 표시하지 마세요
  • • 모호한 에러 메시지를 사용하지 마세요
  • • 인증 과정을 너무 복잡하게 만들지 마세요
  • • 세션 만료 시 데이터를 잃지 않도록 하세요
  • • 자동완성을 차단하지 마세요
