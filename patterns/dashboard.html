Usage Guidelines

Do

  • • 가장 중요한 지표를 상단에 배치하세요
  • • 데이터 변화율을 함께 표시하세요
  • • 일관된 시각적 계층을 유지하세요
  • • 빠른 액션을 위한 바로가기를 제공하세요
  • • 실시간 데이터 갱신을 지원하세요
  • • 반응형 그리드 레이아웃을 사용하세요

Don't

  • • 너무 많은 정보를 한 화면에 담지 마세요
  • • 의미 없는 데이터를 표시하지 마세요
  • • 차트만으로 정확한 값을 파악하게 하지 마세요
  • • 데이터 로딩 상태를 무시하지 마세요
  • • 접근성을 고려하지 않은 색상을 사용하지 마세요
  • • 모바일에서의 사용성을 간과하지 마세요
