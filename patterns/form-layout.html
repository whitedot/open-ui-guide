Usage Guidelines

Do

  • • 관련 필드는 그룹으로 묶어주세요
  • • 필수 필드는 명확하게 표시하세요
  • • 실시간 유효성 검사를 제공하세요
  • • 에러 메시지는 구체적으로 작성하세요
  • • 긴 폼은 단계별로 나누세요
  • • 모바일에서 적절한 키보드 타입을 사용하세요

Don't

  • • 불필요한 필드를 요구하지 마세요
  • • 에러 발생 시 입력 데이터를 지우지 마세요
  • • 라벨 없이 플레이스홀더만 사용하지 마세요
  • • 너무 많은 필드를 한 화면에 표시하지 마세요
  • • 제출 후 피드백 없이 방치하지 마세요
  • • 자동 포맷팅으로 사용자를 혼란스럽게 하지 마세요
