Usage Guidelines

Do

  • • 검색과 필터를 쉽게 접근할 수 있게 배치하세요
  • • 현재 표시 중인 항목 수와 전체 개수를 표시하세요
  • • 빈 상태에서 다음 액션을 안내하세요
  • • 로딩 시 스켈레톤 UI를 사용하세요
  • • 일괄 작업 시 선택 개수를 명확히 표시하세요
  • • 목록/그리드 뷰 전환 옵션을 제공하세요

Don't

  • • 한 페이지에 너무 많은 항목을 표시하지 마세요
  • • 페이지네이션 없이 무한 스크롤만 사용하지 마세요
  • • 검색 결과가 없을 때 빈 화면만 표시하지 마세요
  • • 필터 옵션을 너무 많이 노출하지 마세요
  • • 모바일에서 테이블 레이아웃을 그대로 사용하지 마세요
  • • 정렬 상태를 사용자에게 숨기지 마세요
